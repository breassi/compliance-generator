# Compliance Generator

O **Compliance Generator** é uma ferramenta que automatiza a criação de estruturas e processos de conformidade para projetos de software, permitindo gerar rapidamente políticas, auditorias, relatórios e controles regulatórios para aplicações em .NET, Node.js, Python, Go.

## Tecnologia

- **Linguagem principal:** .NET (C#)  
- **Por que da escolha:** Sistemas de compliance corporativo exigem regras complexas e integração com bancos de dados robustos e auditoria detalhada, onde .NET se destaca.

## Arquitetura

- **Arquitetura recomendada:** Layered Architecture / DDD  
- **Por que da escolha:** Compliance exige regras de negócio complexas e auditoria; arquitetura em camadas garante organização e manutenção.
